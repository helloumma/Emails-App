# Email App

This is a back-end application which sends speaks to a React front-end to send emails.

## 📋 Aims of App

- Allow the user to be able to send emails

## 👩‍💻 Technical Details

- NodeJS
- Nodemailer
- CORs
- Express

## 🔧 How to Run the App

- `npm install`
- `npm start`

## 💭 Future Improvements

- Send emails at a specific date and time (prompted by user on front-end)
